# Sprint 767 — Predictive Release Risk Intelligence

Sprint 767 adds deterministic release risk scoring.

Inputs include:

- change size;
- files changed;
- migration count;
- test coverage;
- failed checks;
- open incidents;
- recent rollbacks;
- deployment frequency;
- production targeting;
- emergency-change status.

Outputs include:

- risk score;
- risk level;
- blocking decision;
- contributing factors;
- remediation recommendations.
