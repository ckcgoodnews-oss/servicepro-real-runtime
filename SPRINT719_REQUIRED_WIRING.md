# Sprint 719 Required Wiring

- Every authenticated dashboard page is wrapped by the shared `AppShell`.
- The shell provides responsive navigation, tenant context, breadcrumbs, theme controls, search, and a profile menu.
- `Ctrl+K` or `Command+K` opens global navigation search; `Escape` closes open menus.
- Mobile navigation uses an accessible menu button, dismissible scrim, and full-height drawer.
- Navigation destinations scheduled for later Phase 46 sprints become live as their route implementations land.
